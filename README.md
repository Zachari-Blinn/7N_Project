## Technologies
Project is created with:
* Vue - version 2.0
* MongoDB as Database
* Mongoose as ORM
* Node
* Express

## Setup
To run this project, install it locally using npm:

```
$ git clone git@github.com:Zachari-Blinn/7N_Project.git
$ cd .\back\ 
$ npm i
$ npm run dev
$ cd .\front\
$ npm i
$ npm run serve

```
